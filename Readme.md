# File upload using Multer+NodeJS

### Installation

Use the package manager npm to install node_modules

```bash
npm install
```
### Starting Server

Use the package manager npm to install node_modules

```bash
node server
```



## Usage

```
/api/file/save : req.file should be file that needs to be saved
/api/file/get/:slug (GET) : get the file which is saved 
```
